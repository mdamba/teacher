# 外国人向け日本語教師募集LP デザイン指示書

この指示書は、LPのデザイン制作を外部（ウェブデザイナー／制作会社）へ依頼する際に、トーン、カラー、構成、各要素の配置を具体化するためのものです。貴社事業の専門性と信頼性が伝わる表現を最優先にします。

## 1. プロジェクト概要

| 項目 | 詳細 |
| --- | --- |
| LPの目的 | 企業向け日本語教育および生活支援事業において、専門性の高い講師を安定的に採用すること |
| ターゲット | 日本語教師養成講座修了者、経験者、特定業種（製造／介護／IT等）に関心のある講師 |
| 訴求ポイント | 安定した業務機会、専門性の向上、柔軟な働き方、企業支援というやりがい |
| LPの目標 | 月間〇〇件の応募獲得（カジュアル面談予約を含む） |

## 2. トーン＆マナー／カラースキーム

| 項目 | 指定内容 | 補足事項 |
| --- | --- | --- |
| 全体印象 | プロフェッショナル、信頼感、清潔感 | カジュアルな求人サイトとは一線を画し、キャリアアップを支援するビジネスパートナーとしての印象を与える。企業向け事業のため、教育的な温かみよりもビジネスとしての信頼性を重視する。 |
| メインカラー | ネイビー（信頼・安定）例：`#003366` | 見出し、重要な背景、キービジュアル周りの要所に使用 |
| サブカラー | アクティブオレンジまたはグリーン（親しみ・行動・活気）例：`#FF6600` または `#339966` | CTAボタン、メリットを強調する要素（数字／アイコン等）に使用 |
| ベースカラー | 白／薄いグレー 例：`#FFFFFF`、`#F5F5F5` | セクションの区切り・背景で可読性を担保 |
| フォント | 日本語：Noto Sans JP または同等の可読性が高いゴシック体 | ウェイト（太さ）を使い分け、階層を明確化 |

## 3. 各セクションの構成とデザイン指示

### 3-1. FV（ファーストビュー）
**見出し案**：【企業の未来と、あなたの専門性をつなぐ】…

- **ビジュアル**：企業の研修室で、講師と外国人社員がプロフェッショナルな雰囲気で交流する写真（自然な笑顔＋真剣さ）。背景に日本の観光風景は避ける。
- **配置**：画面中央に大きくキャッチコピーを配置し、下部にサブカラーの大きなCTAボタンを配置（例：「講師登録・面談エントリーへ」）。

### 3-2. 課題とミッション
**見出し案**：今、企業が求める日本語教師の新しいカタチ

- **表現**：企業の抱える課題（定着率、コミュニケーション不足）を吹き出しやデータ（数値イメージ）で提示。
- **流れ**：課題 → 解決策として貴社事業を提示、というストーリーが視覚的に追える構成。
- **トーン**：落ち着いた青系トーンで信頼性を維持。

### 3-3. 事業紹介
**見出し案**：企業の成長と定着率を支える、ワンストップ・サポート

- **レイアウト**：二本柱（教育／生活支援）を2列で配置。
- **アイコン**：青系のシンプルなアイコン（教育＝本、生活支援＝家 等）で視認性を高める。

### 3-4. 講師のメリット（最重要）
**見出し案**：貴社で働く3つの魅力：安定した環境と確かなキャリア

- **強調**：各メリットをメインカラーのボックス（または円形）で囲み、サブカラーの数字（① ② ③）を大きく配置して視線を集める。
- **補足**：「専門性」では「専門認定制度」のバッジ／証明書を想起させるグラフィックを添えて、キャリア価値を視覚化する。

### 3-5. 募集要項／仕事内容
**見出し案**：あなたの「教える」が企業を変える：募集要項

- **整理**：応募資格などの必須情報は表組み（テーブル）で整理。
- **可読性**：背景色は薄いグレーにして視認性を高める。
- **リスト**：チェックマークのアイコン（サブカラー）で要点を列挙。

### 3-6. 働く環境／サポート体制
**見出し案**：あなたの活躍を後押しする充実のサポート

- **視覚化**：講師管理システム、研修体制、報酬体系をPC／タブレットのモックアップや歯車アイコン等で表現。
- **狙い**：貴社の管理・サポート体制が現代的で効率的であることを訴求する。

### 3-7. 講師の声（あれば）
**見出し案**：「ここで、私のキャリアが変わりました」：講師インタビュー

- **必須要素**：講師の写真を大きめに配置。
- **読みやすさ**：コメントは白抜き（または余白を十分に確保）し、引用符デザインで信頼感を演出。

### 3-8. 応募フォーム／CTA
**見出し案**：まずはカジュアル面談から。エントリーはこちら

- **固定CTA**：スクロール追随の固定フッター型CTAボタン（サブカラー）を検討。
- **フォーム**：目立つように、明るい背景色＋ネイビーの枠線で囲って誘導する。

## 4. 技術的要件

- **レスポンシブ**：モバイル表示を最適化すること（特にCTAはタップしやすいサイズ）。
- **画像**：ファイルサイズを最適化し、表示速度を担保する。
- **SNS連携（フッター）**：企業としての信頼性を担保するため、SNSアイコン（Facebook、Xなど）へのリンク配置を検討する。
